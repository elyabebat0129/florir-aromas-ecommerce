# Florir Aromas E-Commerce

A PHP-based e-commerce system for selling aromas and incense products.

This project was developed for real usage and includes both storefront and administrative features for managing products, clients and orders.

---

## Features

* Product management (CRUD)
* Client management
* Supplier management
* Order management
* Authentication system
* File upload support
* Administrative interface

---

## Tech Stack

* PHP
* MySQL
* JavaScript
* CSS
